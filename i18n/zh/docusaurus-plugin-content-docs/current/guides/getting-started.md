# 快速開始

使用 AiShort 時，您只需將所選提示詞複製到任意 AI 對話模型中，即可獲得符合指引方向的輸出。以下是具體步驟：

1. 搜尋或選擇所需的提示詞，然後點擊提示詞中的「複製」按鈕。

2. 將提示詞貼到 ChatGPT、Gemini 或其他任意 AI 對話模型中。

3. 根據對話提示輸入您的需求。
